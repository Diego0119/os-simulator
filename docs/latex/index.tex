Este proyecto es un Simulador de Sistemas Operativos, diseñado para emular el comportamiento de un sistema operativo básico. A través de esta simulación, puedes explorar cómo se manejan procesos, memoria y otros recursos del sistema. Nuestro enfoque es educativo, para ayudar a estudiantes a entender los conceptos fundamentales detrás de los sistemas operativos.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Funcionamiento}{Funcionamiento}}\label{index_autotoc_md2}
Para poder ejecutar el proyecto de manera satisfactoria, realice los siguientes pasos.


\begin{DoxyEnumerate}
\item En caso de que falten carpetas (src, obj, incs, docs o build), ejecute el comando \textquotesingle{}make folders\textquotesingle{} en la terminal.
\item Asegurece de que los archivos \textquotesingle{}.c\textquotesingle{} estén ubicados correctamente en la carpeta \textquotesingle{}src\textquotesingle{} y que el archivo \textquotesingle{}.h\textquotesingle{} esté en la carpeta \textquotesingle{}incs\textquotesingle{}.
\item Se recomienda limpiar los archivos \textquotesingle{}.o\textquotesingle{} con el comando \textquotesingle{}make clean\textquotesingle{} antes de compilar.
\item Ejecute el comando \textquotesingle{}make\textquotesingle{} en su terminal para poder compilar los archivos \textquotesingle{}.c\textquotesingle{} y crear los archivos \textquotesingle{}.o\textquotesingle{}.
\item Ejecute el comando \textquotesingle{}make run\textquotesingle{} en su terminal para poder comenzar con la prueba del programa Simulador Sistema Operativo.
\item En la carpera \textquotesingle{}os-\/simulator\textquotesingle{} se encuentran dos archivos (.eps y .png) que contienen la carta gantt de los procesos ejecutados con el programa.
\item En caso de querer eliminar los archivos \textquotesingle{}.o\textquotesingle{}, ejecute el comando \textquotesingle{}make clean\textquotesingle{} en su terminal para limpiar las carpetas.
\end{DoxyEnumerate}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Documentación}{Documentación}}\label{index_autotoc_md3}
Antes de generar la documentación, por favor vaya a la carpeta docs para verificar si ya existe la documentación. Para poder generar la documentación del proyecto, realice los siguientes pasos.


\begin{DoxyEnumerate}
\item Asegurece de que esté el archivo \textquotesingle{}config\textquotesingle{} en la carpeta \textquotesingle{}os-\/simulator\textquotesingle{}.
\item Ejecute el comando \textquotesingle{}doxygen config\textquotesingle{} en su terminal para poder ejecutar el archivo config y generar la documentación.
\item En la carpeta docs, se encuentra la documentación en html y pdf. 
\end{DoxyEnumerate}
